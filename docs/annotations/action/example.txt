If a request is routed to a controller without specifying the action, the default action will be executed. You can set the default action for a controller by using the _@action_ annotation. If an _@action_ annotation specified, the default action will be _index_.

In the following example, the default action for the _BookController_ is set to _list_, so that a request to _/book_ executes _/book/list_.

**BookController.cfc**

	/**
 	 * @accessors true
 	 * @action list
	 * @singleton
	 */
	component {

		property _Book;
	
		function list() {

			params.books = _Book.list();
	
		}

	}
