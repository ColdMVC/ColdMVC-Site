In the following example, the default action for the _BookController_ is set to _list_, so that a request to _/book_ executes _/book/list_.

**BookController.cfc**

	/**
 	 * @accessors true
 	 * @action list
	 * @singleton
	 */
	component {

		property _Book;
	
		function list() {

			params.books = _Book.list();
	
		}

	}
