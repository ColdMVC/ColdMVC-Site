Creates a link to a [favicon] [1].

### Parameters

| Name | Required | Type   | Default                      | Description             |
|------|----------|--------|------------------------------|-------------------------|
| url  | false    | string | #linkToImage("favicon.ico")# | The URL of the favicon. |

### Example

	<c:favicon />

[1]: http://en.wikipedia.org/wiki/Favicon "Favicon"
