Creates a [body] [1] tag with event-specific classes. Also outputs any content previously wrapped in the [htmlbody] [2] tag.

[1]: http://www.w3.org/TR/html-markup/input.date.html
[2]: /tags/htmlbody

### Parameters

| Name  | Required | Type    | Default | Description               |
|-------|----------|---------|---------|---------------------------|
| class | false    | string  |         | A class for the body tag. |

### Example

	<html>
		<head>
			<title>This is my page</title>
		</head>
		<c:body>
			<c:render />
		</c:body>
	</html>
