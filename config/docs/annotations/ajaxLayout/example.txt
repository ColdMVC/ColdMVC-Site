In the following example, Ajax requests made to _/book/list_ will_ use the _simple_ layout, while Ajax requests made to _/book/show_ will use the _modal_ layout.

**BookController.cfc**

	/**
 	 * @accessors true
 	 * @ajaxLayout simple
	 * @singleton
	 */
	component {

		property _Book;
	
		function list() {

			params.books = _Book.list();
	
		}
		
		/**
		 * @ajaxLayout modal
		 * /
		function show() {

			params.book = _Book.get(params.id);
	
		}

	}
