In the following example, a _validationService_ exposes its _validate_ method to all models as _validate_.

**ValidationService.cfc**
	
	/**
	 * @accessors true
	 */
	component {

		property validator;

		/**
		 * @modelHelper validate
		 */
		public string function validate(required any model) {
	
			return validator.validateModel(arguments.model);
	
		}

	}
