In the following example, the _setup_ method will be called after both the _configPath_ and _xmlParser_ properties have been injected into the _errorService_ component.

**ErrorService.cfc**

	/**
	 * @accessors true
	 * @initMethod setup
	 * @singleton
	 * /
	component {

		property configPath;
		property xmlParser;

		public void function setup() {

			config = xmlParser.parse(configPath);

		}

		public void function handleError(required any exception) {
	
			if (config.logExceptions) {
				... perform logic here ...
			}

		}

	}
