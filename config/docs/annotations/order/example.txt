In the following example, the default order value for products is set to _asc_.

**Book**

	/**
	 * @persistent true
	 * @extends coldmvc.Model
	 * @sort title
	 * @order asc
	 */
	component {
	
	    /**
	     * @fieldtype id
	     * @generator identity
	     */
	    property id;
	    property title;
	    property author;
	    property description;
	
	}
